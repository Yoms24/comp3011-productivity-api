# COMP3011 Productivity API

A Django-based REST-style API for tracking habits and productivity data.

## Features so far
- Habit model connected to a SQLite database
- Admin interface for managing habits
- Endpoint to list all habits
- Endpoint to retrieve one habit by ID
- JSON responses
- Basic 404 error handling

## Current endpoints
- `/api/habits/`
- `/api/habits/<id>/`

## Tech stack
- Python
- Django
- SQLite
- Git
- GitHub
